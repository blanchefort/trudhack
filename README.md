# Система определения предлагаемой заработной платы

Установка и запуск:

```
git clone https://github.com/blanchefort/trudhack.git
cd trudhack
python -m venv venv
pip install -r requirements.txt
```
Запуск:
```
python app.py
```
Или так:
```
uvicorn app:app --port 5000 --host 0.0.0.0
```
